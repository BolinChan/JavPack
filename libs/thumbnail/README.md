# JavPack.thumbnail

获取预告片

## 内容

| 函数          | 参数           | 注释          |
| :------------ | :------------- | :------------ |
| fetchBlogJav  | `code: string` | BlogJav 爬取  |
| fetchJavStore | `code: string` | JavStore 爬取 |

## 使用

在脚本元属性加入：

- `// @require [request url]`
- `// @require [url]`
- `// @grant GM_xmlhttpRequest`
- `// @connect javstore.net`
- `// @connect blogjav.net`
- `// @connect pixhost.to`
