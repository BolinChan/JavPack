# JavPack.request

网络请求

## 内容

| 函数      | 参数                            | 注释         |
| :-------- | :------------------------------ | :----------- |
| request   | `url: string` `options: object` | 网络请求     |
| taskQueue | `res: string` `steps: func[]`   | 简易任务队列 |

## 使用

在脚本元属性加入：

- `// @require [url]`
- `// @grant GM_xmlhttpRequest`
